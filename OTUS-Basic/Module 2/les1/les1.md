## 1. Типы данных в Rust 

[Типы Данных](https://doc.rust-lang.ru/book/ch03-02-data-types.html#:~:text=%D0%AF%D0%B7%D1%8B%D0%BA%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20Rust-,%D0%A2%D0%B8%D0%BF%D1%8B%20%D0%94%D0%B0%D0%BD%D0%BD%D1%8B%D1%85,-%D0%9A%D0%B0%D0%B6%D0%B4%D0%BE%D0%B5%20%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D0%B5%20%D0%B2)

Rust — строго типизированный язык. Это означает, что каждая переменная, константа, значение выражения имеет определённый тип. Это повышает безопасность программы и позволяет обнаруживать ошибки на этапе компиляции.

Rust поддерживает как примитивные типы данных (числа, булевы значения, символы), так и более сложные (кортежи, массивы).

### 1.1 Примитивные типы

| Категория          | Примеры типов                                                                                                   | Описание                                                   |
| ------------------ | --------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| Целые числа        | `i8`, `i16`, `i32`, `i64`, `i128`, `isize` (знаковые), `u8`, `u16`, `u32`, `u64`, `u128`, `usize` (беззнаковые) | Представляют собой целые числа со знаком или без.          |
| Вещественные числа | `f32`, `f64`                                                                                                    | Числа с плавающей точкой. По умолчанию используется `f64`. |
| Логический тип     | `bool`                                                                                                          | Может принимать значения `true` или `false`.               |
| Символы            | `char`                                                                                                          | Символ Unicode (один символ в одинарных кавычках).         |

**Примеры:**

```rust
let age: u32 = 30;
let temperature: f64 = -5.2;
let is_active: bool = true;
let letter: char = 'R';
```

---

### 1.2 Составные типы

| Тип данных | Пример             | Описание                                                      |
| ---------- | ------------------ | ------------------------------------------------------------- |
| Кортежи    | `(i32, bool, f64)` | Фиксированный набор значений разных типов.                    |
| Массивы    | `[i32; 3]`         | Последовательность элементов одного типа фиксированной длины. |

**Примеры:**

```rust
let person: (i32, &str) = (25, "Alice");
let scores: [i32; 3] = [90, 85, 77];
```

Обращение к элементам:

```rust
let name = person.1;
let first_score = scores[0];
```

## 2. Переменные

В Rust переменные по умолчанию **неизменяемы**. Чтобы сделать переменную изменяемой, используется ключевое слово `mut`.

| Пример объявления | Описание                            |
| ----------------- | ----------------------------------- |
| `let x = 5;`      | Неизменяемая переменная типа `i32`. |
| `let mut y = 10;` | Изменяемая переменная.              |

Пример:

```rust
let mut counter = 0;
counter = counter + 1;
```

Если попытаться изменить значение неизменяемой переменной, компилятор выдаст ошибку.

## 3. Операции присваивания и арифметика

Rust поддерживает основные арифметические и логические операции.

[Переменные и изменяемость](https://doc.rust-lang.ru/book/ch03-01-variables-and-mutability.html#:~:text=%D0%AF%D0%B7%D1%8B%D0%BA%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20Rust-,%D0%9F%D0%B5%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5%20%D0%B8%20%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D1%8F%D0%B5%D0%BC%D0%BE%D1%81%D1%82%D1%8C,-%D0%9A%D0%B0%D0%BA%20%D1%83%D0%BF%D0%BE%D0%BC%D0%B8%D0%BD%D0%B0%D0%BB%D0%BE%D1%81%D1%8C%20%D0%B2)

### 3.1 Арифметические операции

| Операция  | Символ | Пример   | Результат        |
| --------- | ------ | -------- | ---------------- |
| Сложение  | `+`    | `5 + 3`  | `8`              |
| Вычитание | `-`    | `9 - 4`  | `5`              |
| Умножение | `*`    | `2 * 3`  | `6`              |
| Деление   | `/`    | `10 / 3` | `3` (если целое) |
| Остаток   | `%`    | `10 % 3` | `1`              |

### 3.2 Присваивание

```rust
let mut a = 5;
a = a + 2;
```

Можно также использовать сокращённую форму:

```rust
a += 2;
```

## 4. Функции

[Функиции](https://doc.rust-lang.ru/book/ch03-03-how-functions-work.html#:~:text=%D0%AF%D0%B7%D1%8B%D0%BA%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20Rust-,%D0%A4%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D0%B8,-%D0%A4%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D0%B8%20%D1%88%D0%B8%D1%80%D0%BE%D0%BA%D0%BE%20%D1%80%D0%B0%D1%81%D0%BF%D1%80%D0%BE%D1%81%D1%82%D1%80%D0%B0%D0%BD%D0%B5%D0%BD%D1%8B)

Функции в Rust объявляются с помощью ключевого слова `fn`.

### 4.1 Объявление функции без параметров

```rust
fn say_hello() {
    println!("Hello!");
}
```

### 4.2 Функция с параметрами и возвращаемым значением

```rust
fn add(x: i32, y: i32) -> i32 {
    x + y
}
```

| Часть                  | Пример           | Описание                                        |
| ---------------------- | ---------------- | ----------------------------------------------- |
| Имя функции            | `add`            | Название функции                                |
| Аргументы              | `x: i32, y: i32` | Параметры и их типы                             |
| Возвращаемый тип       | `-> i32`         | Тип возвращаемого значения                      |
| Возвращаемое выражение | `x + y`          | Последнее выражение без `;` возвращается неявно |

Если использовать `;` в конце, функция вернёт `()`, то есть ничего.

Пример с `return`:

```rust
fn double(n: i32) -> i32 {
    return n * 2;
}
```

## 5. Вывод типов

Rust способен сам определить тип переменной на основе значения:

```rust
let a = 5;      // Rust определяет a как i32
let b = 3.14;   // b: f64
```

Для точности и читаемости рекомендуется **явно указывать типы**, особенно в публичных API и при работе с коллекциями или обобщениями.

## 6. Константы и статические переменные

### 6.1 `const`

Объявляет **константу времени компиляции**. Она должна иметь **явный тип** и быть **неизменяемой**.

```rust
const MAX_SCORE: u32 = 100;
```

| Характеристика | Значение                                      |
| -------------- | --------------------------------------------- |
| Тип обязателен | Да                                            |
| Инициализация  | Только константными выражениями               |
| Доступность    | Глобальная, без ограничения области видимости |

## Заключение

| Концепция              | Пример                             | Комментарий                               |
| ---------------------- | ---------------------------------- | ----------------------------------------- |
| Объявление переменной  | `let a = 5;`                       | Неизменяемая                              |
| Изменяемая переменная  | `let mut b = 10;`                  | Можно изменить                            |
| Арифметика             | `a + b`, `b % 2`                   | Поддерживаются все базовые операции       |
| Функция                | `fn sum(a: i32, b: i32) -> i32 {}` | Возвращает сумму                          |
| Вывод типа             | `let x = 42;`                      | Тип выводится автоматически               |
| Константа              | `const PI: f64 = 3.14;`            | Неизменяемое значение на этапе компиляции |
